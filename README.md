## USO

```
//suba o docker
docker run --name mon -e MONGO_INITDB_ROOT_USERNAME=admin -e MONGO_INITDB_ROOT_PASSWORD=123 -p 27017:27017 -d mongo:4

// instale as dependecias
npm i

// inicie o servido
npm start


```
